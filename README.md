# Recovery password proyect

Recovery password with email (test)

Tech stack:

- NodeJS
- Expressjs
- Mongo / Mongoose
- Passport / Passport-local
- Nodemailer
- Bcrypt
- JWT
- Handlebars

## Initialize proyect

> Create '.env' file (view .env.example)

```
npm install
```

```
npm start
```
